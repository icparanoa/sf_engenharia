Projeto Hidrossanitários

Um projeto hidráulico consiste em todo o sistema que faz parte da arquitetura do imóvel para abastecer, distribuir e escoar a água. Nele, estão inclusas as redes de distribuição de água fria e quente, caixa d'água, coleta de esgoto e águas pluviais.

Tipos de projetos que a S&F realiza:

  • Projeto de instalações hidráulicas (água fria e quente), esgoto sanitário e pluvial;
  • Projeto de sistemas de reuso de águas pluviais e águas cinzas;
  • Projeto de instalações hidráulicas para piscinas;
  • Projeto de sistema de recalque de esgoto e / ou águas pluviais se necessário;
  • Projeto de sistema de recalque de água potável se necessário;
  • Quantitativo básico dos materiais utilizados.
